### Running Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/kkrishguptaa/artemis.git
   ```

2. Navigate to the project directory:

   ```bash
   cd artemis
   ```

3. Install dependencies:

   ```bash
   pnpm install
   ```

4. Run the development server:

   ```bash
   pnpm dev
   ```
